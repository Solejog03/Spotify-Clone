# 🎵 Spotify Clone

A Spotify-inspired music player built with pure HTML, CSS, and JavaScript — no frameworks, no libraries.

## ✨ Features

- Play, pause, next, and previous controls
- Dynamic playlist loading
- Automatic song fetching from folders
- Interactive seek bar
- Volume control
- Real-time song information
- Responsive design
- Vanilla JavaScript only

## ▶️ Running Locally

1. Clone the repository
2. Open live server on localhost

## 📂 How to add your songs

While some sample songs are there if you want to make your own playlist and add your songs it is simple.

1. Create a new folder inside the `songs` directory. (It will be playlist)
2. Add your audio files / songs to that folder.
3. Refresh the application.
4. add a cover of your choice to it and name it cover
5. copy info.json from other premade playlist and paste it and change name and description to your choice

The playlist and its songs will be loaded automatically.
